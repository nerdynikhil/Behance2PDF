Behance2PDF is a production-grade Chrome extension for creative professionals, students, and design enthusiasts who need to **save a Behance project as a PDF** or **download Behance images in bulk**. With a single click, export any Behance project as a beautifully formatted PDF, or bulk-download every image at its highest available resolution — so you always have offline access to the projects you care about.

🌐 **Website:** [www.nerdynikhil.com/behance2pdf](https://www.nerdynikhil.com/behance2pdf)

---

## 🚀 Features

- **One-Click PDF Export:** Instantly save Behance projects as high-quality PDFs, preserving original layouts and image fidelity.
- **Bulk Image Download:** Download all project images as a ZIP archive, with smart, meaningful file naming.
- **High-Resolution Assets:** Always get the best quality images available from each project.
- **Seamless Integration:** Non-intrusive UI that appears only on Behance project pages.
- **Robust Error Handling:** Clear feedback and graceful recovery from network or page issues.
- **Privacy-First:** No data collection, tracking, or external analytics—your browsing stays yours.

## 🛠 Installation

Behance2PDF isn't on the Chrome Web Store yet (see [FAQ](#-faq)), so for now it's installed manually as an unpacked extension:

1. Grab the latest [Release](https://github.com/nerdynikhil/Behance2PDF/releases) ZIP (or download this repo as a ZIP via the green **Code** button → **Download ZIP**).
2. Unzip it to a folder you'll keep around (don't delete it after installing—Chrome loads the extension from this folder).
3. Open `chrome://extensions` in Chrome.
4. Enable **Developer mode** (toggle in the top-right corner).
5. Click **Load unpacked** and select the unzipped folder.
6. Pin the extension for quick access (optional).
7. Visit any Behance project page and start saving!

## 📖 Usage

1. Navigate to a Behance project on [behance.net](https://www.behance.net/)
2. Click the floating **Save as PDF** or **Download Images** button
3. Your files will be automatically downloaded with project-based names

## ❓ FAQ

**Is Behance2PDF free?**
Yes — completely free and open source, no account or sign-up required.

**Why isn't it on the Chrome Web Store?**
It's pending publication. The code has already been updated to address Chrome Web Store review feedback; until the listing is live, install it manually using the steps above.

**Does it work with all Behance project layouts?**
Yes — it's built to handle every Behance project type, including image galleries, text sections, and embedded media.

**Can I download images in their original resolution?**
Yes — bulk image download always grabs the highest resolution available for each asset.

**Is my data tracked?**
No. Behance2PDF collects no data and includes no analytics or tracking — see the [Privacy Policy](https://www.nerdynikhil.com/behance2pdf/privacy).
